    • A (MG): When launching Jupyter Lab from a terminal, the directory currently active in the terminal (what is displayed when you run pwd) determines which directory the Jupyter Lab interface shows in the browser (and you are limited to navigating within subdirectories of this directory).   • Q: What is the difference between “” and ‘’?

    • A (MG): In Python both single-quotes ' and double quotes " can be used to delimit strings and there is no difference in how they are interpreted. The only rule is that you use the same type of quote character to end the string as you use to start. It can be sometimes useful to use one particular type when representing a string which includes the other type, for example "This is a string with a 'quote' in it", as this avoids needing to escape the quote characters within the string.

  • Q:In question 5 of Socrative, can I use “numpy.mean(numpy.std(data))”?

    • A (AR): numpy.std(data) calculates the standard deviation over the whole array. So you get a single number, which applies to all patients for all days. Doing a mean on that single number will just give you the same number back. But the question is asking you to calculate the SD per patient, and then get the mean of those SDs. So you need numpy.mean(numpy.std(data, axis=1)). If you try both options, you should get different answers.

  • Q:Could you please explain the two numbers A and B in “figsize = (A,B)”?

    • A (TC): These are the width (A) and height (B) of the overall figure (in inches). You’ll see in Q7 if you try to rearrange the plots so they’re above each other, they appear squashed. You’d need to change the figsize so its height is about 10, and width is about 3.5 to fix this. The ‘type’ of the figsize argument is called a tuple. This is similar to a list, but is immutable and can be written as a list inside parentheses; e.g. mytuple = (1,2,3)
    • A (MG): As an additional tip, if you need to quickly check what arguments are available for a particular function and how they should be specified in Jupyter Lab you can run a cell with the name of the command followed by ? to get the documentation for the function, for example plt.figure? will show the documentation for plt.figure which includes a description of the figsize argument. You can also use help(plt.figure) which will also work in a standard Python interpreter.
